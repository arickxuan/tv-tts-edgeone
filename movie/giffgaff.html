Giffgaff OAuth PKCE 生成工具

步骤 1: 生成参数

步骤 2: 构建授权URL

步骤 3: 登录并获取授权码

在授权页面登录后，浏览器会尝试打开 giffgaff://auth/callback/?code=XXXXXX&state=XXXXXX

从浏览器地址栏复制完整的回调URL，提取 code 参数值

步骤 4: 将参数填入Postman

将以下值填入Postman的集合变量中：

  • code_verifier ← 步骤1中的Code Verifier
  • code_challenge ← 步骤1中的Code Challenge
  • oauth_state ← 步骤1中的State
  • auth_code ← 步骤3中的Auth Code

然后在Postman中执行 "步骤2: 交换Token" 请求！
